# MULimnology_Sites
Overview of all MULimnology site metadata

This repository will have all site data for all Missouri sites

All site information, including MULakeNumber, waterBody name (and older names), samplingSite names, and latitude and longitude of the sampling sites are here: 
  "01_MainData/MULimnology-MULakeNumber-samplingSite-Main.csv"
New sites/MULakeNumbers will be added here along with lats/longs and sampling site names and locations - they can be added manually.

Also, metadata about the specific lakes and reservoirs will be updated here.
  "01_MainData/MULimnology-MULakeNumber-waterBodyCharacteristics-Main.csv"
The waterbody specific latitude and longitude are based on the dam location - where outflow is. The watershed characteristics were orginally run by Dr. Lorena Silva and methods can be found here: https://portal.edirepository.org/nis/metadataviewer?packageid=edi.1830.2
Not all waterbodies have complete waterbody and watershed characteristics but they all have lat/longs.
